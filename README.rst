
hoppy
==============

hoppy is a Python library for accessing the Hoptoad API.

.. _Hoptoad: http://hoptoadapp.com/


Requirements
------------

hoppy requires:

* Python_ 2.6
* A Hoptoad_ account

.. _Python: http://python.org/


Installation
------------

hoppy is available on PyPi, and the recommended method of installation is pip::
    
    pip install hoppy
